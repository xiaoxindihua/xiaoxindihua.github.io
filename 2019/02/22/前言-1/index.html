前言

发表于 2019-02-22

2017年10月末，参加了林明樟（MJ）老师的超级数字力课程，MJ老师在短短的两天时间里，将看似复杂的财报知识以轻松活泼的方式教授给大家。很多人买股票要么是凭感觉，要么是凭消息、甚至是小道消息，却很少有人是通过查看公司的财务报告来购买股票的，究其原因，并不是投资者不想通过看财报来投资，而是财报确实看起来太复杂，资产负债表、利润表、现金流量表，一大堆的数字，看着就头疼，还有天书一样的审计意见，让投资者根本无从下手。两天的课程学下来，发现财务报表跟很多知识一样，都是一层窗户纸，只是如果没有人帮你捅破，就一直没办法摸到门道。

学习完毕，一腔热血，觉得通过股票赚钱也并非难事，可是被各种事一拖，也就渐渐失去了当初的动力。为了能够巩固所学，准备每天解读一只股票，其实也算不上解读，只是按照MJ老师教的方法，选出自己认为值得投资的股票，即是对所学知识的复习，也是对投资过程做个记录。

分析股票的数据主要来源于财报说和雪球。
